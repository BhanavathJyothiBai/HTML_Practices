Multimedia Tags

  • programmers prefer multimedia tags in order to provide advertisement on the browser.
  • There are three multimedia tags are as follows:-
    1. Image
    2. Video
    3. Audio
  1. < img > :- it is used for provide images on the browser.
    2. it takes four attributes are as following:-
    • src: it stands for source attribute which is basically used to provide image filename/path.
    • width & height:They are used to control the dimension of an image.
    • alt :it stands for "alternate text" used to specify some kind of message if given file name does not exists.
      • For Example:- < img src="img.extension" width="100" height="100" alt="not found">< /img >
  2. < video > & < audio > : it is used to specify video files and audio files on the browser.The common attribute between tags are "src" and "controls" where control is used display pause, play buttons for the audio & video. To play video automatically on webpage make use of "autoplay" attribute with "muted". For Example:- < video src="video.extension" controls>< /video >
