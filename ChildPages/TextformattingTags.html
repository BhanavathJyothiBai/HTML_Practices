Text Formatting tags

Move to Homepage

  1. < em >: it stands for emphasis tag.it is used to make the text contents appear in italic format on the web page.for example: < em > This is italic < /em >.
  2. < strong >: it is used to represent text contents in bold format on the web page.
  3. < ins >: it stands for insertion tag.To give the underline for the text content on the web page.
  4. < del >: it stands for deletion tag.it is used to give the strike on the text contents. for example: strike
  5. < sup > </sup>: sup stands for superscript which is used to provide the contents above the text. for example: 23=8.
  6. < sub > </sub>: sub stands for subscript which is used to provide the contents below the text. for example: H2O.
  7. < hr > tag : it is used to draw horizontal line on the webpage.
  8. < center > < /center > : it is used to represent the content at the exact center of the web browser.
  9. Headings:
    1. To provide Headings on the browser programmers using heading tags.
    2. By using following tags we can provide different kinds of headings.
      • < h1> < /h1 >
      • < h2 > < /h2 >
      • < h3 > < /h3 >
      • < h4 > < /h4 >
      • < h4 > < /h5 >
      • < h6 > < /h6 >
    3. < h1 > gives the largest font-size which is possible in html and also make the contents to appear as bold whereas < h6 > gives the smallest font-size possible in html.
    4. From < h1 > to < h6 > font-size is decreasing but it makes the contents to appear as bold.
  10. < pre > tag : it stands for preformatted text. it is used to consider html file spaces on web pages.
  11. < mark > : it is used to highlight the text content by providing yellow background color as(default).
