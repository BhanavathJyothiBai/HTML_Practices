HTML List

  1. Programmers prefer HTML files concept to provide different form of "bullets"for the text contents.
  2. To list down the html content or text content on the web browser, programmers prefer HTML lists.
  3. HTML lists are two types:-
    1. Ordered list
    2. Unordered list
      3.
  4. Ordered list:- It is represented by "< ol >" and on the browser in the form of "numbers".
  5. UnOrdered list:- It is represented by "< ul >" and on the browser in the form of "solid sphere".
  6. The text which is associated along with the form of bullets is technically referred as "list item"and it is represented by using "< li >".
  7. As a programmer we can change bullets or remove them by using "CSS" propertiescalled it as "list-style-type"for it we can pass different values mentioned as below:-
    1. For UnOrdered list programmer can pass value as:-
      • Disc
      • Circle
      • Square values
        •
    2. For Ordered list programmer can pass value as:-
      1. Decimal
      2. Upper-alpha
      3. Lower-alpha
      4. Upper-roman
      5. Lower-roman values
        6.
    3. To remove bullets from < ul > and < ol > programmer can pass value as "none" value.
  8. For Example:-
    • < ul style="list-style-type:circle;" >
    • < li > first item < /li >
    • < li > second item < /li >
    • < /ul >
