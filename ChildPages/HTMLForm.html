HTML FORM

  1. As a programmer we used html forms to take values from user into the application.
  2. Programmatically html forms consists two important components:
    1. Label
    2. Input box
  3. Labels are the directions provided for users to enter appropriate data where input is the actual data which is provided by the user.
  4. Labels are indicated by using < label > < /label > and to get data from the user we have to create.
    5. input-box and it is represented by using < input > < /input >.
  5. To create HTML forms we have to use < form > < /form >
  6. For example:-
    • < form >
    • < label > mobile number < /label >  
    • < input > < /input >
    • < br > < /br >
    • < /form >
  7. To create watermark on the input box we make use of "placeholder" attribute with < input >.
  8. To make fields mandatory on web page in html form with input tag make use of "required"attribute.


    9. S.No: Input type: Behavior: Example:
    1 < input type="text" > User can enter alphabets,numbers,and special symbols.
    2 < input type="number" > User can enter only numbers.
    3 < input type="password" > User can enter alphabets,numbers,and special symbols but all will be replace with solid sphere.
    4 < input type="radio" > User will be given with radio button which results in single selection. MaleFeMale
    5 < input type="checkbox" > User will be given with checkbox which results in multiple selections. Programming Web Technologies Core Java
    6 < input type="date" > User can give the data in the format of Date dd-mm-yyyy and also gets calender view.
    7 < input type="month" > User will get the calender with only month and year.
    8 < input type="time" > User will get the option to enter the time with different values.
    9 < input type="email" > User will enter the valid pattern of email id in the input box.
    10 < input type="file" > User will get option to uploading the file.
    11 < input type="submit" > User can submit the form by clickling on it.
    12 < input type="reset" > User can clear the values which entered in the form inputs
