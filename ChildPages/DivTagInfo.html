Div Tag

< div > < /div >:

  1. This tag is used to create different sections on the web-browser for navigation bar,advertisement display the information,etc...
  2. To work with div tag programmers have to make use of "CSS" properties like height,width,background-color and float.
  3. Height & Width are used to change the dimensions of div tag.
  4. Background-color is used to provide background for div tag.
  5. Float is used to fix div tag position.
  6. For Float "CSS" property we can pass two values--> left & right.
    7. For Example:-
    • < div style="height:100px; background-color:blue; float:left;">
    • < h1 > section on the browser < /h1 >
    • < /div >
